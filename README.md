# OrangeHRM Manual Testing

A comprehensive manual testing project for the OrangeHRM web application, demonstrating the complete Software Testing Life Cycle (STLC), including test case design, test execution, defect management, and reporting.

## 📌 Project Overview

This project focuses on validating the core functionalities of the OrangeHRM application through structured manual testing. Test cases were created in Microsoft Excel, executed against the application, and defects were reported and tracked using Jira.

## ✅ Testing Activities

- Requirement Analysis
- Test Scenario Creation
- Manual Test Case Design
- Functional Testing
- Smoke Testing
- Sanity Testing
- Regression Testing
- Defect Reporting & Tracking using Jira
- Bug Retesting
- Test Summary Reporting

## 📂 Modules Tested

- Login
- Dashboard
- PIM (Employee Management)
- Leave Management
- Admin

## 🛠 Tools & Technologies

- OrangeHRM
